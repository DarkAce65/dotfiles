### Dependency list

- [ly](https://github.com/fairyglade/ly) - Display manager
- [hyprland](https://hypr.land/) - Desktop environment
- [dms](https://danklinux.com/) - DankMaterialShell, desktop shell
- [kitty](https://sw.kovidgoyal.net/kitty/) - Terminal
- [zim](https://github.com/eriner/zim) - Shell

##### Extras

- vim with [pathogen](https://github.com/tpope/vim-pathogen) and [lightline](https://github.com/itchyny/lightline.vim) - Terminal editor
- [pfetch](https://github.com/Un1q32/pfetch) - Sysinfo script

### Installation

- Install [hyprland](https://hypr.land/)
- Install [Dank Linux](https://danklinux.com/)
- Copy the following files and folders to their respective destinations
  - **.config** folder
  - **.zshenv** > ~/
